# シフカレ Live

シフトボードの文章を貼り付け、メンバー全員のシフトを月間カレンダーで共有するWebアプリです。

## 主な機能

- メールアドレスと6桁コードによるパスワードなし認証
- 登録済みメンバーだけが閲覧可能
- ランチ・ディナー・通し勤務の色分け
- 閲覧モードと変更モード
- 全メンバーのシフト追加・変更・削除
- シフトボード文章の一括登録
- リアルタイム同期
- 変更履歴の自動保存
- スマートフォン対応

## 初期設定

1. Supabase Dashboard の `SQL Editor` を開く
2. `supabase.sql` の全文を貼り付けて実行する
3. Authentication の Email Provider を有効にする
4. メールテンプレートに `{{ .Token }}` を含め、6桁コードが届くようにする
5. Vercelでこのリポジトリをデプロイする

Supabaseの公開用URLとPublishable Keyは `config.js` に設定します。`service_role` キーなどの秘密鍵はブラウザ側へ置かないでください。
