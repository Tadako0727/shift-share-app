# シフカレ Live

メールOTP認証、Supabaseリアルタイム同期、シフトボード文章の一括登録に対応したシフト共有アプリです。

## 初回設定

1. Supabase Dashboardで対象プロジェクトを開く。
2. `SQL Editor` → `New query` を開く。
3. `supabase.sql` の全文を貼り付けて `Run` を押す。
4. `Authentication` → `Providers` → `Email` を有効にする。
5. メールテンプレートをOTPコード式にする場合は、本文に `{{ .Token }}` を含める。
6. GitHubリポジトリ直下に、このフォルダ内のファイルをアップロードする。
7. VercelでリポジトリをImportし、Framework Presetは `Other`、Root Directoryは `./` のままDeployする。

## ホスト

初期ホストは `kowita51@gmail.com` です。初回ログイン後、左側の「メンバー管理」から、利用を許可するメールアドレスと表示名を追加できます。

## セキュリティ

- 未登録メールアドレスで認証ユーザーが作成されても、RLSによりプロフィール・シフトは閲覧できません。
- 登録済みかつ有効なメンバーのみ、シフトを閲覧・編集できます。
- メンバー登録・無効化はホストだけが行えます。
- シフト変更履歴は自動記録されます。
